# Java Implementation Navigator

这是一个 VS Code 扩展，用于在 Java 代码中快速导航到接口的实现类。

## 功能

- 提供快捷命令用于跳转到 Java 接口的实现类
- 支持通过命令面板或快捷键快速跳转

## 使用方法

1. 在 Java 接口或抽象类的代码中，使用以下方式之一：
   - 右键菜单选择"跳转到实现"
   - 使用命令面板（Ctrl+Shift+P）输入"跳转到实现"
   
## 要求

- VS Code 1.97.0 或更高版本
- Java 开发环境
